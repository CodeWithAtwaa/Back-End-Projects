## 🚀 Getting Started

### - Using Docker 
### - Prerequisites

* PHP 8.x or later
* Composer
* MySQL
* Apache / Nginx
* Git

### Installation

```bash
git clone https://github.com/CodeWithAtwaa/Back-End-Projects.git

cd Back-End-Projects
```

Navigate to the desired project folder and follow its setup instructions.

For Laravel projects:

```bash
composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
